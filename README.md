# webrtc-build

1、支持拉取不同版本
2、支持环境变量参数
3、支持单纯编译
4、源码拉取按平台和版本分类存放
5、编译后按平台和版本分类输出

#### 支持的系统
- windows x86_64
- macos_arm64
- macos_x86_64
- raspberry-pi-os_armv6 (Raspberry Pi Zero)
- raspberry-pi-os_armv7 (Raspberry Pi 3, 4)
- raspberry-pi-os_armv8 (Raspberry Pi 3, 4)
- ubuntu-18.04_armv8
    - Jetson Nano
    - Jetson Xavier NX
    - Jetson AGX Xavier
- ubuntu-18.04_x86_64
- ubuntu-20.04_x86_64
- android
- ios
